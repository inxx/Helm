Agents

Agent 바이너리 경로와 인증은 .helm/config.json에서 관리합니다. Phase 1에서는 UI 편집을 제공하지 않습니다.

설정 파일 위치 <repo>/.helm/config.json
codex
/opt/homebrew/bin/codex 기본값. config의 agentBinaries.codex로 덮어쓸 수 있습니다.
claude
/opt/homebrew/bin/claude 로컬 인증 필요. 401 발생 시 claude CLI 인증을 확인하세요.
gemini
/opt/homebrew/bin/gemini 환경변수 > config > 기본값 순서로 적용됩니다.

Workflow

커밋 체크 명령과 PR base 브랜치 기본값입니다.

defaultCheckCommand
npm run check --check 옵션이 없으면 commit 시 이 명령을 실행합니다.
prBaseBranch
main --base 옵션이 없으면 이 값으로 PR을 생성합니다.
